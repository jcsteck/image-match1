  • Test different image files with, only, one pixel with different pixel info in one of the images.
  • Test different image files tolerance is set to 80% of matching. Only one different pixel.
  • Test different image files with different image info and show the diff in one new canvas.
  • Test different image files tolerance is set to 80% of matching. The different image has 50% of different pixel info.
